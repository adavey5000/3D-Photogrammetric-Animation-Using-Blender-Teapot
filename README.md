# 3D-Photogrammetric-Teapot-Animation

# Demo 
CSC322finalAnimationGitHub.mp4

# Description
A 20-second animation of a real-world teacup created using photogrammetry with PolyCam and animation with Blender.

School project for CSC 322 - Introduction to Computer Graphics

# Concepts
- Photogrammetry techniques
- Creating a mesh using PolyCam
- Animation principles
- Blender UV editing, texturing, and motion rendering and animation
